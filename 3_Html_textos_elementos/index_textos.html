Textos

O HTML permite que exista marcações de textos de diferentes formar.

  • Parágrafos: Utilize a tag <p> para criar Parágrafos.
  • Parágrafos Utilize as tags <h1> a <h6> para enfatizar os textos.
  • Texto em negrito Utilize as tags <strong> ou <i> para destacar o testo em itálico.
